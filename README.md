Paratrooper on arcade-henkinen 2D-peli, jossa pelaaja ohjaa tykkiä 
ja yrittää estää laskuvarjosotilaita sekä helikoptereita laskeutumasta
maahan. Tavoitteena on selviytyä mahdollisimman pitkä
än kasvavaa vihollisaaltoa vastaan.




ohjaimet:
pelin käynnistys: paina alku näytössä hiiren vasenta npäppäintä.
tykin liikuttaminen: liikuttamalla hiirtä.
sarja tuli: spacebar
ampuminen: pidä hiiren vasenta näppäintä pohjassa.


pisteytys:
punanen laskuvarjo 100p
sininen laskuvarjo 300p
keltanen laskuvarjo 500p

sininen laatikko: isommat ammukset
keltanen laatikko: 1000p
oranssi laatikko: bursti
vihreä laatikko: 3x ammukset
punainen laatikko: yksi elämä lisää
violetti laatikko: hidastaa vihollisia 50%
kirkkaan punanen laatikko: tuhoo kaikki laskuvarjot
keltanen laatikko: 2x pisteet
